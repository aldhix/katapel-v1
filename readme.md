# Katapel

Katapel adalah singkatan dari Katalog Aplikasi, berbasis web dengan menggunakan framework Laravel. 

## Menggunakan Framework

 - Laravel 5.8  
 - Font Awesome 
 - Chart.js

## Fitur
### Halaman Guest :

 1. Home
 2. Kategori
 3. Search / Pencarian
 4. Detail Produk
 5. About
 6. Order / Pesan

### Halaman Admin :

 1. Login
 2. Dashboard
 3. Kategori
 4. Produk
 5. Pesanan
 6. Pengguna / User
 7. Profil Pengguna
